# 项目启动指南

## 推荐方式：只启动Flask后端

### 1. 启动Flask后端（必需）
```bash
# 激活虚拟环境
.\.venv\Scripts\Activate.ps1

# 设置环境变量
$env:FLASK_ENV = "development"

# 启动Flask服务器
python backend\wsgi.py
```

### 2. 访问应用
- 主页面：http://localhost:5000/
- 登录页面：http://localhost:5000/login.html
- Dashboard：http://localhost:5000/dashboard.html

## 可选方式：同时运行前端服务器

如果你想要更好的开发体验，可以同时运行两个服务：

### 终端1：Flask后端
```bash
.\.venv\Scripts\Activate.ps1
$env:FLASK_ENV = "development"
python backend\wsgi.py
```

### 终端2：前端服务器
```bash
cd frontend
python -m http.server 8080
```

然后访问：http://localhost:8080/

## 故障排除

### 如果登录失败：
1. 确保Flask后端正在运行（端口5000）
2. 检查浏览器控制台是否有CORS错误
3. 访问 http://localhost:5000/api-test.html 测试API连接

### 如果页面无法加载：
1. 检查Flask后端是否正常启动
2. 确认端口5000没有被其他程序占用
3. 查看终端是否有错误信息
